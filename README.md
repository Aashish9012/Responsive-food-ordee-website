Responsive Food Ordering Website

A modern, fully responsive **Food Ordering Website** built using **HTML, CSS, and JavaScript**.
This project allows users to browse food items, add them to a cart, and place orders with an interactive and user-friendly interface.

Features

*  Fully responsive design (Mobile, Tablet, Desktop)
*  Dynamic food menu with categories
*  Add to Cart / Remove from Cart functionality
*  Quantity selector for each item
*  Real-time total price calculation
*  Search and filter food items
*  Dark mode toggle
*  Toast notifications (item added/removed)
*  Cart data stored using Local Storage

Technologies Used

* **HTML5** – Structure of the website
* **CSS3** – Styling (Flexbox, Grid, Media Queries)
* **JavaScript (ES6)** – Logic and interactivity
* **Local Storage API** – Data persistence
